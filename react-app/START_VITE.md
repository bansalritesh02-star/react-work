# Starting This Vite App

Follow these steps from a terminal.

## 1. Go to the project directory

```bash
cd /Users/ritesh/my-projects/react2/react-work/react-app
```

## 2. Install dependencies, if needed

This project already has a `node_modules` folder, so dependencies appear to be installed.

If you ever clone or copy the project fresh, run:

```bash
npm install
```

## 3. Start the Vite development server

```bash
npm run dev
```

This runs the `dev` script from `package.json`:

```json
"dev": "vite"
```

## 4. Open the app in the browser

After the server starts, Vite will print a local URL in the terminal.

It is usually:

```text
http://localhost:5173/
```

Open that URL in your browser to view the React app.

## 5. Stop the server

When you are done, go back to the terminal running Vite and press:

```text
Control + C
```
